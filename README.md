# タイピングゲーム

ブラウザで動くシンプルな日本語・英語タイピング練習ゲームです。

## 遊び方

1. `index.html` をブラウザで開く
2. 難易度を選ぶ（かんたん / ふつう / むずかしい / English）
3. 入力欄をクリックして文字を入力するとタイマーがスタート
4. 60秒間でどれだけ正確に速く入力できるか挑戦！

## 機能

- WPM（1分あたりの単語数）リアルタイム表示
- 正確さ（%）表示
- 60秒カウントダウンタイマー
- 難易度別ハイスコア保存（ブラウザのローカルストレージ）
- 正解・ミス文字のハイライト表示

## 難易度

| 難易度 | 内容 |
|--------|------|
| かんたん | ひらがな単語 |
| ふつう | ひらがな文章 |
| むずかしい | 漢字混じりの早口言葉 |
| English | 英語フレーズ |
